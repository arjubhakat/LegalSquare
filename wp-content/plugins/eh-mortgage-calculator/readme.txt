== Description ==

EH Mortgage Calculator is a simple plugin designed to calculate a mortgage payment.
The plugin creates an amortization table, and pagination for easier readability.
The plugin also provides a widget for use in your sidebar

== Installation ==

1. Upload `eh-mortgage-calculator` unzip folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode [ehmortgagecalculator] in a page or post.
1. Or add the widget to your sidebar.

== Frequently Asked Questions ==

= Can I adjust the settings? =

No settings are provided in this release.  However, future releases will provide the 
ability to customize the input fields and values.

= Is the plugin internationalized? =

Yes.  The plugin is ready for localization.  A translation to Spanish is already provided.
